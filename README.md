<h1>TradeMe-Manager App</h1>



<h2>How To Use</h2>

Login with your API consumer key and consumer secret key. 

Manage your trademe listing

<h2>Development</h2>

1. copy the code and clone
2. `npm install`
3. create `.env` file
4. define the variable `NODE_MAILER_PASSWORD`
6. `npm run dev`
7. enjoy the App
